## 关于Picamera

Picamera是使用Python开发的图传监控系统，基于树莓派和普通web摄像头硬件。

系统包括运行于树莓派上的图像采集socket客户端;运行于linux系统下的socket服务器;linux系统下的http服务器。

之后的版本将增加图像跟踪和识别功能。

[系统说明](http://oliverobot.com/github_file/picamera01.png])

### Dependencies

python 2.x/3.x

opencv3
